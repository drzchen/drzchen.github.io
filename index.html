I'm a Professor at the College of Computer Science and Technology, Nanjing University of Aeronautics and Astronautics, China. I received my PhD degree in Computer System from LAAS-CNRS, INSA de Toulouse, France. I am a member of the Association for Computing Machinery (ACM), a distinguished member of the China Computer Federation (CCF), and a member of the CCF technical committee on formal methods.

Research Interests and Publications

I'm interested in:
  • Software engineering, software verification, incl. model checking and runtime verification
  • Programming languages, program semantics, program analysis
  • Theoretical computer science, formal methods, automata and formal languages theory
  • Applications of software verification

Tools

We have developed the following tools:
  • Movec: an automated tool for runtime monitoring and verification of C programs as an extension to the C programming language.
  • Monic: a checker for LTL formulas.
  • Smc: an automated Scade model checker for verifying Scade programs.
